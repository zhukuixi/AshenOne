# Lecture 8. Solving Ax = b: row reduced form R
* Complete Solution for Ax=b: X = Xp+Xn 
* Rank r
* if r=m(行数）,solution exists.
* if r=m(行数)=n(列数）,solution is unique.  
* Xp为特解，求解时将自由变量全赋值0.  
* Xn为null space的向量。可以按照L7求解.
