# Lecture 25: Symmetric matrices and Positive Definiteness
* 对称矩阵eigenvalue为实数，eigenvector can be chosen as互相垂直  
* 对称矩阵pivot的乘积=其eigenvalue的乘积=det(A) 
* 正定对称矩阵的三大正性质：  
    1.All pivot > 0
    2.All eigenvalue > 0
    3.All sub-determinants > 0
