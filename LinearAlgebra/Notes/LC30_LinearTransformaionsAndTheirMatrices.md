# Lecture 30: Linear transformations and their matrices
* Linear transformation的两个性质：
   1. T(v+w) = T(v)+T(w)
   2. T(c*v) = c\*T(v)
* 如何寻找Linear transformation其后的矩阵  
	A的第i列，是对第i个old basis的线性转换的结果由new basis线性表示的系数。也即T(Vi)在新基W线性表示的系数.(Wj的系数对应Aji)
